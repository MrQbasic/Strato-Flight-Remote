# Strato-Flight-Remote

Remote control and live telemetry access module for **Strato-Flight** — a high altitude balloon firmware platform.

This project is designed as a portable ground station for receiving telemetry, displaying live flight data, and enabling remote commands and WiFi-based data access.

🔗 Related Project - [Strato-Flight](https://github.com/MrQbasic/Strato-Flight)
